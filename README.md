# check-in.html

Лендинг премиального коттеджного поселка в Тетеево (Республика Татарстан) с акцентом на лидогенерацию, SEO/GEO и AI-поиск.

## Что реализовано

- Одностраничный сайт `check-in.html` с блоками: Hero, преимущества, о проекте, локация, инфраструктура, галерея, планировки/цены, FAQ, лид-форма.
- SEO-основа: `title`, `description`, `keywords`, корректные H1-H3.
- GEO-фокус: Казань, Тетеево, Республика Татарстан.
- Микроразметка Schema.org (`RealEstateAgent`, `FAQPage`).
- CTA-кнопки и форма захвата для заявок.

## Запуск

Откройте файл `check-in.html` в браузере.
